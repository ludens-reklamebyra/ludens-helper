# ludens-helper

## Install
```
$ git clone https://github.com/ludens-reklamebyra/ludens-helper.git
$ npm install
```
## Running locally for development
```
$ npm run dev
```
## Build
```
$ npm run build
```
## Tests
Uses mocha with Chai to test with.
```
$ npm test
```
